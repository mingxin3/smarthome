# SmartHomeMonitorQt6 (Base)

- Qt6 Widgets + CMake
- TCP (simulated device server on 127.0.0.1:45454)
- SQLite (schema/seed under assets/db, copied/initialized at first run)
- Operation history + alarm history + CSV export

## Default login
- username: admin
- password: admin

## Build
Open the folder in Qt Creator (CMake project), configure Qt6 kit, build & run.
